# Rent A Car Project
Kodluyoruz DRC Java Bootcamp(Kodla, Kariyerine Ba≈üla Java Bootcamp) graduation project. <br/>
In this Rent A Car Management Project, you can access cars, damage records, customers, invoices, and payments. You can add, search and list each of them.

## Technologies
- SpringBoot DevTools 
- Spring Web
- Spring Data JPA
- H2 Database
- Lombok
- Model Mapper
